# chat-app


# 1.「アプリのタイトル」
　chat with claude3
 
# 2. アプリの説明40字程度
　会話記録をfire baseに保存できるチャットアプリ
  回答ボタンを押すことで、今までの会話の流れを考慮して生成AIが返信してくれる機能をつけました
 
# 3. 頑張った機能やポイント
　fire baseのログをAWS bedrockにPOSTする機能
  生成AIのアウトプットをwebページにコメントとして追記する機能
 
# 4. 課題の感想
aws bedrockとの連携は比較的簡単に行えた
もっと高度なことを生成AIを使って機能提供できるようにしたい
